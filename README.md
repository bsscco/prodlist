# prodlist
상품목록을 내려받아 보여주고 상품을 좋아요, 검색할 수 있는 앱

# 사용 기술
- 싱글 액티비티
- MVI & AAC
- Kotlin(with Coroutine, Flow)
- Compose, Compose Navigation
- 클린 아키텍쳐
- UseCase
- Repository
- 멀티 모듈
